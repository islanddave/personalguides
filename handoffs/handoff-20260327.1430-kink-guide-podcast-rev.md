foot icon uses fixed 36pt Drawing (not size param).
Because: ReportLab LEFTPADDING does not apply to Drawing flowables in Table cells. Margin baked into Drawing coordinates. Do not resize without retesting in-PDF.

lock icon is a padlock, not a rope coil.
Because: ReportLab Polygon fill rule fills the entire convex hull — annulus/ring shapes using Polygon are not possible. Two previous rope-coil attempts both produced filled circles. Padlock replaced it permanently.

All icons were isolation-tested with renderPM at s=26 before committing to script.
Because: Table rendering at 26pt differs from isolated render. Test first, commit after.

Always upload from Mac or re-read from filesystem MCP before building.

ReportLab annulus limitation.
